# 💻 Flappy Bird - Zelda

Este repositório contém o projeto final desenvolvido para a disciplina de **Introdução à Programação**, como parte da grade do curso de **Ciência de Dados e Inteligência Artificial** da **UFPB**.

## 📝 Sobre o Projeto

> **Nota:** O projeto é um jogo de plataforma infinita, onde o jogador controla um personagem que deve desviar de uma série de tubos gerados aleatoriamente. O objetivo é sobreviver o maior tempo possível para alcançar a maior pontuação. 

As mecânicas incluem controle de física (gravidade e pulo), detecção de colisão, sistema de pontuação dinâmico e transições entre estados de jogo. Durante o desenvolvimento, a equipe consolidou o uso da biblioteca Pygame para manipulação de eventos, além de aplicar conceitos estruturais como modularização do código.

O objetivo principal deste projeto foi aplicar na prática os conceitos fundamentais de lógica de programação, laços de repetição, condicionais e estruturação de código utilizando a linguagem Python.

## 🚀 Tecnologias Utilizadas

* [Python 3.x](https://www.python.org/)
* [Pygame 2.6.1](https://www.pygame.org/)

## ⚙️ Como Executar na Sua Máquina

Siga os passos abaixo para testar o jogo localmente:

1. Clone este repositório:
   ```bash
   git clone https://github.com/salazarantonio/flappy-bird.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd flappy-bird
   ```

3. Instale a versão específica da biblioteca Pygame:
   ```bash
   pip install pygame
   ```

4. Execute o arquivo principal do jogo:
   ```bash
   python main.py
   ```
